  • Graph Matching for Microservice Resource Scheduling.

    We try to abstract both hardware resources in cloud environment and microservice tasks as graph and then we will use graph matching algorithm to improve the resource scheduling efficiency with respect to QoS of users.

  • Simulator for Microservice in Cloud Environment

    We focus on both the management of IT and non-IT resource and QoS of micorservice in cloud environment. We build our simulator on the basis of Cloudsim and pay attention to the resource scheduling of microservice. Our goal is to build a simulator for analysis and management of micorservice in cloud.

Selected Projects
  1. Android Memory Managemnt [Code]

    In the first part, I wrote new system calls to map a target process's page table into the current process's address space. Based on that, I implemented VATranslate to get physical address of given virtual address. In the second part, I modified the Linux Page Replacement Algorithm by adding new variables reflecting page importance.

  2. PCC-Vivace Reproduction [Code]

    Based on the PCC-Vivace in NSDI'18, we reproduce this online-learning congestion control algorithm as the course project for Computer Network. We implemented it as a linux module based on the previous work, making it closer to the paper. Then I use mininet to perform small-buffer-test and multi-flow-test to show the efficiency beyond classic TCP congestion control algorithms.
